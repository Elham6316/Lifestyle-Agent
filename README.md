#  Lifestyle Agent🌴

Lifestyle Agent is a personalized AI assistant built using Claude Projects.

The goal of the project is to help users make better use of their free time by providing personalized entertainment and lifestyle recommendations.

---

## Features

- 🎬 Movie recommendations
- 📺 TV show recommendations
- 📚 Book suggestions
- 🎧 Podcasts
- 🎮 Games
- 🏕 Outdoor activities
- 📍 Places to visit
- 🗺 Weekend planning
- ✈ Vacation ideas

---

## How it works

The agent first asks a few simple questions such as:

- Available time
- Energy level
- Indoor or outdoor
- Budget
- Solo or group
- Interests

It then creates personalized recommendations based on the user's preferences.

---

## AI Personality

Lifestyle Agent is designed to be:

- Friendly
- Relaxed
- Fun
- Conversational
- Supportive

The assistant speaks like a close friend instead of a traditional chatbot.
